ABOUT THE MOD
...................................................
3D Torches MOD FOR MINETEST BY INFINATUM
Makes Minetest have 3D torches  instead of 2d
Thank you for using the mod

HOW TO INSTALL?
....................................................

1. extract the "3d_torches" folder
2. Move the "3d_torches" to the mod folder in your mintest game files
3. Start Minetest
4. under the "Singleplayer" tab press the Configure tab
5. enable "3d_torches" then press Save
6. enjoy the mod
